# 这个项目是什么？
中文编程即将崛起，提前整理一些术语，便于大家阅读他人的中文代码

# 怎么用？
如果你的代码里打算用中文汉字来代替英文，可以考虑用本项目里列出的汉字关键词

# 如何添砖加瓦
